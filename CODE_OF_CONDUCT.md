# Code of Conduct

## Our Standards

This is a small utility project. Participation is expected to be respectful and constructive.

Examples of acceptable behavior:
- Reporting bugs with clear reproduction steps
- Proposing enhancements with rationale
- Reviewing code with specific, actionable feedback
- Asking questions about how the schedule logic works

Examples of unacceptable behavior:
- Personal attacks or insults in issues or pull requests
- Dismissing contributions without explanation
- Spam or off-topic promotion

## Scope

This code of conduct applies to the GitHub repository: issues, pull requests, comments, and any other project spaces.

## Reporting

If someone is behaving unacceptably, contact the project author at the email listed on the GitHub profile for [@sanchez314c](https://github.com/sanchez314c).

## Enforcement

Violations may result in comments being removed or contributors being blocked from the repository, at the maintainer's discretion.
